# SPARTA Save Editor

Редактор сохранений для игры **SPARTA 2035**.

Открывает, просматривает и редактирует JSON-файлы сохранений.
GUI на PyQt6.

## Установка

```bash
pip install -r requirements.txt
python main.py
```

## Использование

1. Запустить `python main.py`
2. Файл → Открыть → выбрать .json файл сохранения
3. Редактировать поля
4. Файл → Сохранить
