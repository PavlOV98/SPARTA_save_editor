Редактор сохранений и конфигурационных файлов для игры **SPARTA 2035**.

![Python](https://img.shields.io/badge/python-3.9%2B-blue)
![PyQt6](https://img.shields.io/badge/PyQt-6-green)

## Возможности

| Вкладка | Описание |
|---|---|
| 📁 Выбор файла | Выбор папки с сохранениями, папки с игрой, файла локализации |
| 🌍 Глобальные параметры | Редактирование валюты (деньги, разведданные, размер команды, лимит персонажей) и отношений с фракциями (Front, TsaiShen, Shield) |
| 👤 Редактор персонажей | Все параметры бойцов: уровень, опыт, здоровье, характеристики, экипировка, способности, статистика |
| 📦 Склад | Редактирование storedEquipment с поиском и сортировкой |
| ⚙ Оборудование | Редактирование ItemModuleConfig.json (разделение на Спарта/Враги, синхронизация дублей) |
| 🔫 Оружие | Редактирование WeaponModuleConfig.json (разделение на Спарта/Враги, синхронизация дублей) |
| 🏪 Строения | Редактирование CharacterMetaStatuses.json (бустеры, статусы, лечение) |

## Установка

```bash
git clone https://github.com/PavlOV98/SPARTA_save_editor.git
cd SPARTA_save_editor
pip install -r requirements.txt
python main.py
```
